Cluster Control Commands¶

Below is the list of Cluster Control Language (CCL) commands used for MemSQL operations and maintenance.

Aggregators¶

  • AGGREGATOR SET AS MASTER
  • SHOW AGGREGATORS

Leaves¶

  • ADD LEAF
  • ATTACH LEAF
  • ATTACH LEAF ALL
  • DETACH LEAF
  • REMOVE LEAF
  • SHOW LEAVES

Partitions¶

  • ATTACH PARTITION
  • COPY PARTITION
  • CREATE PARTITION
  • DETACH PARTITION
  • DROP PARTITION
  • EXPLAIN REBALANCE PARTITIONS
  • PROMOTE PARTITION
  • REBALANCE PARTITIONS
  • RESTORE REDUNDANCY
  • SHOW CLUSTER STATUS
  • SHOW PARTITIONS
  • SHOW REBALANCE STATUS

Database Maintenance¶

  • BACKUP
  • REPAIR DATABASE
  • RESTORE
  • SNAPSHOT DATABASE

Replication¶

  • CONTINUE REPLICATING
  • PAUSE REPLICATING
  • REPLICATE DATABASE
  • SHOW REPLICATION STATUS
  • STOP REPLICATING
