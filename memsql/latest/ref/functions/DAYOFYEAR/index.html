DAYOFYEARÂ¶

Extracts the day of the year from a date.

Syntax

DAYOFYEAR (dateobj)

Arguments

  • dateobj: a valid date, datetime, or parsable date string.

Return Type

Integer. If dateobj is not a valid date, returns NULL.

Examples

memsql> select dayofyear('2014-04-18');
+-------------------------+
| dayofyear('2014-04-18') |
+-------------------------+
|                     108 |
+-------------------------+
