SHOW TABLE STATUSÂ¶

Show tables status information in a MemSQL database.

SHOW TABLE STATUS [{FROM | IN} db_name]

Notes

  • db_name is the name of a MemSQL database.
  • Shows how much memory this table is using for row data.
  • Memory for off-row data (such as VARCHARs) is not reported.
  • This command can be run on any MemSQL node (see Node Requirements for MemSQL Commands).

Engine Engine

Row_format Row format
Rows Rows

Collation Collation

Comment Comment
BuffMgr Memory Use Buffer manager memory use (in bytes).
Average GC Duration (ms) Average GC duration (in milliseconds)
