Data Definition Commands¶

Below is the list of Data Definition Language (DDL) commands used to define metadata in MemSQL.

Databases¶

  • CREATE DATABASE
  • DROP DATABASE

Tables¶

  • CREATE TABLE
  • DROP TABLE
  • ALTER TABLE
  • CREATE INDEX
  • DROP INDEX

Views¶

  • CREATE VIEW
  • DROP VIEW
  • ALTER VIEW

Administrative Commands¶

  • USE
  • DESCRIBE
  • DROP ... FROM PLANCACHE
  • SET GLOBAL
  • SELECT GLOBAL
  • KILL CONNECTION
  • KILLALL QUERIES
  • LOCK TABLES
  • UNLOCK TABLES
  • FLUSH TABLES

System Status¶

  • SHOW DATABASES
  • SHOW PLANCACHE
  • SHOW STATUS
  • SHOW STATUS EXTENDED
  • SHOW TABLE STATUS
  • SHOW TABLES
  • SHOW VARIABLES
  • SHOW PROCESSLIST
  • SHOW WARNINGS
  • SHOW COLUMNS
  • SHOW GRANTS
  • SHOW INDEX
  • SHOW INDEXES
  • SHOW KEYS
  • SHOW SCHEMAS
  • SHOW GLOBAL STATUS
  • SHOW GLOBAL VARIABLES
  • SHOW SESSION STATUS
  • SHOW SESSION VARIABLES
