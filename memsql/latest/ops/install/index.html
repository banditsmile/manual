Installing and Configuring MemSQL Ops¶

Important

This documentation applies to MemSQL Ops 4.0 and above. To upgrade your version of Ops, please download the latest version. Enterprise customers can also send an email to support@memsql.com

With the latest version of MemSQL, installing and configuring MemSQL is tightly integrated with MemSQL Ops. As such, all the steps needed to install MemSQL Ops are also the steps in the Installation Guide. This section will not attempt to repeat the information in that section; it will just point to relevant content in the Installation Guide.

MemSQL Ops can be installed online or offline. The best user experience is achieved with installing MemSQL Ops online.

Online Installation¶

To perform a fully online installation, you need to have the following:

  1. a working internet connection through which MemSQL Ops can download MemSQL database binaries and validate license keys
  2. SSH connectivity between MemSQL nodes in a cluster
  3. sudo access on each of the nodes

If you meet these conditions, online MemSQL and MemSQL Ops installation is covered in the Deploying Using MemSQL Ops section of the Quick Start guide. Refer to that section for more information.

If your system does not meet one or more of the above conditions, or if you run into any installation or configuration issues as you set up MemSQL Ops, consult the Offline Installation and the Installation Best Practices sections for more information.

Offline Installation¶

To perform an offline installation, refer to the Offline Installation section of the Installation Guide.

Installation Best Practices¶

To learn more about MemSQL Ops and MemSQL installation best practices, refer to the Installation Best Practices section in the Installation Guide.
