jquery.ui.1.7.2的特效介绍

blind(百叶窗)

Effect

1.1. mode: 用来设定一个特效的动作方式
	1.1.1. show: 显示
	1.1.2. hide: 隐藏
	1.1.3. toggle: 切换显示和隐藏
	1.1.4. 默认hide
	1.1.5. 如果设置为其他任何不接受值, 会先执行动作, 然后显示.
1.2. direction: 设定特效的方向
	1.2.1. vertical: 垂直方向
	1.2.2. level: 水平方向.
	1.2.3. 默认vertical
	1.2.4. 其他任何不接受值, 以水平方向替代.

scale(按比例缩放)

Effect

10.1. mode: 用来设定一个特效的动作方式
	10.1.1. show: 显示
	10.1.2. hide: 隐藏
	10.1.3. toggle: 切换显示和隐藏
	10.1.4. 默认effect
	10.1.5. 如果设置为其他任何不接受值, 会先执行动作, 然后显示.
10.2. percent: 百分比, 指缩放百分之多少之后完全消失,   设置比100小时, 是缩小, 设置比100大时, 是扩大. 默认是0
10.3. direction: 缩放时候的方向….默认是both
	10.3.1. vertical: 仅垂直方向
	10.3.2. horizontal: 仅水平方向
	10.3.3. both: 垂直和水平方向同步.
10.4. origin: 设定每个方向上的缩放起源接受一个两个元素的数组, 第一个元素代表垂直方向上的缩放起源, 第二个参数代表水平方向上的缩放起源,  这个地方源码中没有做容错处理, 哪怕direction设置了只在一个方向缩放, 也需要设置一个数组.   并且, 这个属性只有在mode不为effect的时候才有效.   默认是[‘middle’, ‘center’]
	10.4.1. 垂直方向:
		10.4.1.1. top
		10.4.1.2. middle
		10.4.1.3. bottom
	10.4.2. 水平方向
		10.4.2.1. left
		10.4.2.2. center
		10.4.2.3. right
10.5. from: 对象{width: 120, height: 200}, 有两个属性width和height, 设置的是缩放开始的时候的大小…..默认: mode为show的时候{width: 0, height: 0}, 其他情况下为元素原始大小.

shake(晃动)

Effect

11.1. mode: 用来设定一个特效的动作方式
	11.1.1. show: 显示
	11.1.2. hide: 隐藏
	11.1.3. toggle: 切换显示和隐藏
	11.1.4. 默认effect
	11.1.5. 如果设置为其他任何不接受值, 会先执行动作, 然后显示.
11.2. direction: 晃动的方向, 设置这个属性的源代码是: (direction == 'up' || direction == 'down') ? 'top' : 'left';   可以看出, up和down代表是上下晃动, 其他所有的值都是左右晃动.   但经过测试还是发现up和down有细微区别(决定这种区别的源代码是: var motion = (direction == 'up' || direction == 'left') ? 'pos' : 'neg';  这句代码认为up, left是正向的, 其他的都是负方向的.),  up第一次晃动是向上, down则是向下.  因此有下面结论:
	11.2.1. left: 默认, 从左向右滑动
	11.2.2. up: 从上向下滑动
	11.2.3. down: 从下向上滑动
	11.2.4. 其他任何值: 从右向左滑动
11.3. distance: 晃动的距离.  默认是20
11.4. times: 晃动的次数, 默认是3
11.5. duration: 特效过程持续时间, 单位是毫秒.   其实这里是有点怪异的, 其他特效中都没有加这个”加强处理”(因为duration是统一在外部设置的),  源码是这样的: var speed = o.duration || o.options.duration || 140;  可以看出, 如果在外部设置了duration, 那么这里设置的会失效.  但是, 经过测试, 里面设置而外面不设置会发生错误.  那这里其实就是画蛇添足了.

size(改变大小)

Effect

12.1. mode: 用来设定一个特效的动作方式
	12.1.1. show: 显示
	12.1.2. hide: 隐藏
	12.1.3. toggle: 切换显示和隐藏
	12.1.4. 默认effect
	12.1.5. 如果设置为其他任何不接受值, 会先执行动作, 然后显示.
12.2. to: 对象{width: 120, height: 120}, 这个属性必须设置, 指显示的时候, 最终渲染成多大.
12.3. from: 对象{width: 20, height: 20}, 指开始时的大小, 默认是元素的原始大小.
12.4. origin: 这个size的特效其实还是改变大小, 说白了, 还是缩放.  所以, 也需要通过这个选项来控制各个缩放方向上的源位置, 默认是[‘top’, ‘left’]
	12.4.1. 垂直方向:
		12.4.1.1. top
		12.4.1.2. middle
		12.4.1.3. bottom
	12.4.2. 水平方向
		12.4.2.1. left
		12.4.2.2. center
		12.4.2.3. bottom
12.5. scale: 缩放方式, 在这个size的缩放过程中, 实际上给指定元素的外面套了一个div的(box), 之所以这样, 是因为to选项设置的值可能比要缩放的元素大. 这个时候, 加入一个外框, 会多一种选择, ……可供的选择有:
	12.5.1. box: 缩放外面的box
	12.5.2. content: 缩放我们自己的元素
	12.5.3. both: 两者都缩放…..默认值
12.6. restore: 在size设定的缩放动作完成之后, 是否将元素还原为原始大小.  接受的是布尔类型的值.  默认是false

13.2.1. left: 默认, 从左向右滑动

13.2.3. down: 从下向上滑动

Effect
