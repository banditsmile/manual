activate(事件类型:dropactivate)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
可接受的任意draggable开始活动(被拖动了)
示例:
初始化时设置
$('.selector').droppable({ activate: function(event, ui) { ... } });
动态绑定事件
$('.selector').bind('dropactivate', function(event, ui) { ... });
deactivate(事件类型:dropdeactivate)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
可接受的任意draggable的拖动停止了
示例:
初始化时设置
$('.selector').droppable({ deactivate: function(event, ui) { ... } });
动态绑定事件
$('.selector').bind('dropdeactivate', function(event, ui) { ... });
over(事件类型:dropover)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
可接受的任意draggable到达了当前droppable的上面
示例:
初始化时设置
$('.selector').droppable({ over: function(event, ui) { ... } });
动态绑定事件
$('.selector').bind('dropover', function(event, ui) { ... });
out(事件类型:dropout)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
可接受的任意draggable从当前droppable上面出去了
示例:
初始化时设置
$('.selector').droppable({ out: function(event, ui) { ... } });
动态绑定事件
$('.selector').bind('dropout', function(event, ui) { ... });
drop(事件类型:drop)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
一个可接受的draggable控件放到了当前droppable中, 注意, 这里是放入成功, 也就是说到达了通过tolerance选项设置的认为over的位置并放开鼠标
示例:
初始化时设置
$('.selector').droppable({ drop: function(event, ui) { ... } });
动态绑定事件
$('.selector').bind('drop', function(event, ui) { ... });
注意这里是和hoverClass, tolerance一起结合使用的, 效果比较明显. 这里设置的tolerance是touch, 所以, 只要你的draggable进入droppable一点的时候, 松开鼠标, 就已经表示放入成功了.
显示演示示例
hover draggable
hover_droppable
<style type="text/css">
#msg{color: red; font: 12px;}
.draggable{ width: 70px; height: 40px; padding: 0.5em; float: left; margin: 10px 10px 10px 0; }
.droppable{ width: 150px; height: 150px; padding: 0.5em; float: left; margin: 10px; }
.ui-state-highlight{background: #88EE88;}
</style>
<script type="text/javascript">
$(function() {
$('#hover_draggable').draggable();
$('#hover_droppable').droppable({
tolerance: 'touch',
hoverClass: 'ui-state-highlight',
drop: function(event, ui) {
$('#msg').text($('#msg').text() + '放进去咯');
}
});
});
</script>

</body>
