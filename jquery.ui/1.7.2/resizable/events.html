start(事件类型:resizestart)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
开始拖动改变大小时触发.
示例:
初始化时设置
$('.selector').resizable({ start: function(event, ui) { ... } });
动态绑定事件
$('.selector').bind('resizestart', function(event, ui) { ... });
译者注: 关于resizable的三个事件, 统一做个示例在这里, 运行示例, 一目了然

#resizable_event { width: 150px; height: 150px; padding: 0.5em; }

<script type="text/javascript">
$(function() {
$("#resizable_event").resizable({
start: function(event, ui) {
$('#msg_resizable_event').html($('#msg_resizable_event').html() + '<br />start事件');
},
resize: function(event, ui) {
$('#msg_resizable_event').html($('#msg_resizable_event').html() + '<br />resize事件,宽度' + ui.size.width);
},
stop: function(event, ui) {
$('#msg_resizable_event').html($('#msg_resizable_event').html() + '<br />stop事件');
}
});
});
</script>

resize(事件类型:resize)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
拖动过程中, 鼠标每移动1像素触发一次.
示例:
初始化时设置
$('.selector').resizable({ resize: function(event, ui) { ... } });
动态绑定事件
$('.selector').bind('resize', function(event, ui) { ... });
stop(事件类型:resizestop)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
停止拖动时触发
示例:
初始化时设置
$('.selector').resizable({ stop: function(event, ui) { ... } });
动态绑定事件
$('.selector').bind('resizestop', function(event, ui) { ... });
