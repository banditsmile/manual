文档内容
概览:
$(".selector").animate(styleObject, [duration])
jQuery UI特效的核心扩展了animate功能, 可以很好的动态改变颜色. 大量的使用了类迁移特性可以支持以下的属性的颜色动态改变(CSS样式):
backgroundColor
borderBottomColor
borderLeftColor
borderRightColor
borderTopColor
color
outlineColor
设置改变的时候, 可以接受的颜色值:
十六进制: #FFFFFF
RGB颜色: rgb(255, 255, 255)
颜色名称: "white"

调用代码示例:
$(".selector").animate({
backgroundColor: "#DDDDDD",
borderBottomColor: "rgb(66, 89, 198)",
color: "green",
width: 500
}, 2000);

参数:
styleObject: 对象类型, 上面列举的可以用来改变的颜色属性以该对象的属性出现. 值可以用上面列出的任意一种方式进行设置. 同时, 对于其他CSS样式的动态切换, 也可以作为这个对象的属性的方式进行设置.
duration: 整个动态改变的切换过程持续的时间. 可以是slow, normal, fast, 或指定具体的时间数值(单位是毫秒.), 默认值是normal.
