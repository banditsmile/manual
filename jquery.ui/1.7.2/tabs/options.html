ajaxOptions(默认值:null)
可选类型:
对象
描述:
加载tab内容时Ajax请求的选项, 请参见jQuery的ajax.
译者注: 由于tabs自身就对ajax请求做了一些封装, 所以在添加ajaxOptions时请对照源码, 以免造成冲突.
示例:
初始化时设置
$('.selector').tabs({ ajaxOptions: { async: false } });
获取选项值
var ajaxOptions = $('.selector').tabs('option', 'ajaxOptions');
设置选项值
$('.selector').tabs('option', 'ajaxOptions', { async: false });
cache(默认值:false)
可选类型:
布尔值
描述:
是否缓存远程tabs内容, 也就是说仅加载一次远程内容还是每次点击tabs的时候都从远程加载. 缓存的内容通常是有延迟的, 即仅第一次点击一个tab的时候才会真正从远程加载.
译者注: 这里的cache选项会屏蔽ajaxOptions中的cache选项, 因此, 不要试图从ajaxOptions中设置缓存.
示例:
初始化时设置
$('.selector').tabs({ cache: true });
获取选项值
var cache = $('.selector').tabs('option', 'cache');
设置选项值
$('.selector').tabs('option', 'cache', true);
collapsible(默认值:false)
可选类型:
布尔值
描述:
设置为true之后, 再次选择当前选中的标签, 会隐藏内容.
示例:
初始化时设置
$('.selector').tabs({ collapsible: true });
获取选项值
var collapsible = $('.selector').tabs('option', 'collapsible');
设置选项值
$('.selector').tabs('option', 'collapsible', true);
cookie(默认值:null)
可选类型:
对象
描述:
把最后选择的tab存放到cookie中. 如果没有定义selected选项, 那么在一次加载时终端就会首先从cookie中初始化应该选择哪个tab.
要使用这个选项需要增加jquery的cookie插件(http://plugins.jquery.com/project/cookie).
这个选项要求的是对象, 对象的属性应该是cookie插件中期望的属性. 可用的属性有{expires: 7, path: '/', domain: 'jquery.com', secure: true}.
jquery.ui 1.7之后, 还支持通过使用name属性定义cookie的名称.
示例:
初始化时设置
$('.selector').tabs({ cookie: { expires: 30 } });
获取选项值
var cookie = $('.selector').tabs('option', 'cookie');
设置选项值
$('.selector').tabs('option', 'cookie', { expires: 30 });
deselectable(默认值:false)
可选类型:
布尔值
描述:
jquery.ui 1.7开始将这个选项置为不建议使用的. 用于collapsible.
译者注:
下面是deselectable选项的处理源码:
if (this.options.deselectable !== undefined) {
this.options.collapsible = this.options.deselectable;
}
仅仅是将deselectable有值时, collapsible使用它的值.
示例:
初始化时设置
$('.selector').tabs({ deselectable: true });
获取选项值
var deselectable = $('.selector').tabs('option', 'deselectable');
设置选项值
$('.selector').tabs('option', 'deselectable', true);
disabled(默认值:[])
可选类型:
数组
描述:
数值数组, 数组中描述的索引对应的tab在初始化后被设置为失效的.
译者注:
tab的索引是按照定义时的顺序, 从0开始的有序数字.
比如, 定义了
<ul>
<li>标签A</li>
<li>标签B</li>
<li>标签C</li>
</ul>
那么对应的索引就是0, 1, 2.
如果要使B和C初始化时失效, 那么设置disabled: [1, 2]就可以了
示例:
初始化时设置
$('.selector').tabs({ disabled: [1, 2] });
获取选项值
var disabled = $('.selector').tabs('option', 'disabled');
设置选项值
$('.selector').tabs('option', 'disabled', [1, 2]);
event(默认值:click)
可选类型:
字符串
描述:
用来选择tab的事件类型.
译者注: 这里需要注意的是传递的是html原生事件名称. 比如, mouseover, mouseout
示例:
初始化时设置
$('.selector').tabs({ event: 'mouseover' });
获取选项值
var event = $('.selector').tabs('option', 'event');
设置选项值
$('.selector').tabs('option', 'event', 'mouseover');
fx(默认值:null)
可选类型:
对象
数组
第一个元素表示显示时的动作选项. 第二个元素表示隐藏时的动作选项.
描述:
允许动态的隐藏和显示tab面板. duration选项可以是一个字符串('slow', 'normal', 'fast')或者数字(单位是毫秒.) 默认值为normal.
示例:
初始化时设置
$('.selector').tabs({ fx: { opacity: 'toggle' } });
获取选项值
var fx = $('.selector').tabs('option', 'fx');
设置选项值
$('.selector').tabs('option', 'fx', { opacity: 'toggle' });
idPrefix(默认值:ui-tabs-)
可选类型:
字符串
描述:
生成id的前缀.
如果有远程tab, 它的锚点元素(也就是tab的dom定义中的<a />标记), 没有使用title属性来生成id, 那么就会使用这里指定的前缀和$.data(element)生成的id组成的字符串作为这个tab的panel的id属性的值.
示例:
初始化时设置
$('.selector').tabs({ idPrefix: 'ui-tabs-primary' });
获取选项值
var idPrefix = $('.selector').tabs('option', 'idPrefix');
设置选项值
$('.selector').tabs('option', 'idPrefix', 'ui-tabs-primary');
panelTemplate(默认值:<div></div>)
可选类型:
字符串
描述:
通过add方法或远程加载tab内容创建新的tab面板时使用的外部标签.
示例:
初始化时设置
$('.selector').tabs({ panelTemplate: '<li></li>' });
获取选项值
var panelTemplate = $('.selector').tabs('option', 'panelTemplate');
设置选项值
$('.selector').tabs('option', 'panelTemplate', '<li></li>');
selected(默认值:0)
可选类型:
数字
描述:
从0开始的tab序列表示的是初始化时候默认选中的tab索引. 如果要设置所有的tab都不被选中, 可以将selected选项设置为-1.
示例:
初始化时设置
$('.selector').tabs({ selected: 3 });
获取选项值
var selected = $('.selector').tabs('option', 'selected');
设置选项值
$('.selector').tabs('option', 'selected', 3);
spinner(默认值:<em>Loading&#8230;</em>)
可选类型:
字符串
描述:
在加载远程tab内容的过程中, 该tab的标题将显示为spinner选项指定的HTML内容, 传递一个空字符串就是阻止这种行为.
译者注:
以下是spinner选项的处理源码:
var span = $('span', a);
span.data('label.tabs', span.html()).html(o.spinner);
第一句中的a就是tabs定义里面的a标签封装后的jquery对象, 可以看出, 是从a的子元素中查找了span标签处理的, 所以, 无论如何, tab的a内部一定要有span标签spinner才是有效的.
示例:
初始化时设置
$('.selector').tabs({ spinner: 'Retrieving data...' });
获取选项值
var spinner = $('.selector').tabs('option', 'spinner');
设置选项值
$('.selector').tabs('option', 'spinner', 'Retrieving data...');
tabTemplate(默认值:<li><a href="#{href}"><span>#{label}</span></a></li>)
可选类型:
字符串
描述:
一个tab被创建或添加的时候用来解析它的模板. 占位符#{href}和#{label}分别指url和tab的标题, 在add方法中以参数的方式传递.
译者注:
以下为tabTemplate处理的源码:
$(o.tabTemplate.replace(/#\{href\}/g, url).replace(/#\{label\}/g, label))
读出tabTemplate的字符串之后, 将#{href}和#{label}分别进行了替换.
另外一点需要注意的是, 在自定义的模板中, 也请保留一个a标签和一个span标签的嵌套(请参见spinner选项.)
示例:
初始化时设置
$('.selector').tabs({ tabTemplate: '<div><a href="#{href}"><span>#{label}</span></a></div>' });
获取选项值
var tabTemplate = $('.selector').tabs('option', 'tabTemplate');
设置选项值
$('.selector').tabs('option', 'tabTemplate', '<div><a href="#{href}"><span>#{label}</span></a></div>');
