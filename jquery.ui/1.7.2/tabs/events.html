select(事件类型:tabsselect)
回调函数参数:
event
ui
ui.panel: jquery封装后的选择的tab的面板对象
ui.tab: jquery封装后的选择的tab标签
ui.index: 选择的tab的索引.
详细:
当通过event选项指定的事件发生时(选择一个tab), 触发.
示例:
初始化时设置
$('.selector').tabs({ select: function(event, ui) { ... } });
动态绑定事件
$('.selector').bind('tabsselect', function(event, ui) { ... });
