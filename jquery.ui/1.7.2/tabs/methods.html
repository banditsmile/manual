destroy
详细描述:
完全移除一个tabs控件, 使其回退到该元素被初始化成tabs控件之前的状态.
示例:
$(".selector").tabs('destroy');
disable
详细描述:
将tabs控件样式改变成为失效, 与enable对应.
示例:
$(".selector").tabs('disable');
enable
详细描述:
将tabs控件样式改变为允许, 与disable对应.
示例:
$(".selector").tabs('enable');
option
参数:
optionName(字符串)
选项一节中列举的所有选项名都可用.
value(任意)
该选项可以接受的任意数据值.
详细描述:
获取或设置tabs控件的选项, 第二个参数是选项名, 第三个参数是值. 如果不指定值, 就是获取, 指定值, 就是设置.
示例:
设置选项的值
$(".selector").tabs('option', optionName[, value]);
add
参数:
url(字符串)
tab的面板中显示的内容链接, 可以通过id选择器#的方式指定页面内已有的元素, 也可以指定一个相对或绝对的url地址.
label(字符串)
tab的标题
index(数字)
新的tab要被插入的位置.
详细描述:
添加一个新的tab
示例:
$(".selector").tabs( 'add' , url , label , [index] );
remove
index(数字)
要删除的tab的索引位置.
详细描述:
删除指定索引的tab.
示例:
$(".selector").tabs( 'remove' , index );
enable
index(数字)
要使之有效的tab的索引位置.
详细描述:
使指定索引的tab有效.
一次是所有当前失效的tab都有效, 使用$('#example').data('disabled.tabs', []);
示例:
$(".selector").tabs( 'enable' , index );
disable
index(数字)
要使之失效的tab的索引位置.
详细描述:
使指定索引的tab失效.
一次使多个tab失效使用$('#example').data('disabled.tabs', [1, 2, 3]);
示例:
$(".selector").tabs( 'disable' , index );
cancel
index(数字)
要选择的tab的索引位置, 或者要选择的tab面板的id选择器.
详细描述:
选择一个tab, 就和它被点击(指event选项设置的事件发生)的效果一样.
示例:
$(".selector").tabs( 'select' , index );
load
index(数字)
要重新加载的tab的索引.
详细描述:
重加载一个ajax tab的内容. 这个方法总是从远程加载内容, 哪怕cache设置为true.
示例:
$(".selector").tabs( 'load' , index );
url
index(数字)
要改变url的tab的索引
url(字符串)
新的url地址
译者注: 这里url设置为相对或绝对的路径都没有问题, 但是设置为页内的id选择器有问题.
详细描述:
改变指定索引的tab的远程加载路径, 注意, 这里不仅仅是可以改变远程加载tab的url, 也可以将原本指向页内内容的tab改变为远程地址. 这个url的改变并不会马上加载, 而是等到下一次选中该tab时加载.
示例:
$(".selector").tabs( 'url' , index , url );
length
详细描述:
匹配的tab的数量.
示例:
$(".selector").tabs( 'length');
abort
详细描述:
终止当前正在运行的所有的ajax请求和动态效果
示例:
$(".selector").tabs( 'abort');
rotate
ms(数字)
以毫秒为单位, 表示每两次切换之间的时间间隔.
continuing(布尔型)
可选参数.
表示在自动切换过程中, 用户手动选择了一个tab之后, 是否还继续自动切换
详细描述:
是tabs开始自动切换. 每隔ms毫秒切换一次.
示例:
$(".selector").tabs( 'rotate' , ms , [continuing] );
