change(事件类型:progressbarchange)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
processbar的value发生改变
示例:
初始化时设置
$("#progressbar").progressbar({ change: function(event, ui) { //this等价于event.target alert('又完成了1%'); } });
动态绑定事件
$("#progressbar").bind('progressbarchange', function(event, ui) { //this等价于event.target alert('又完成了1%'); });
