start(事件类型:dragstart)
回调函数参数:
event
原生的浏览器事件.
ui
JQuery的ui对象
详细:
拖动开始, 指鼠标按下, 开始移动
示例:
初始化时设置
$('.selector').draggable({ start: function(event, ui) { alert(this); }, });
动态绑定事件
$(".selector").bind('dragstart', function(event, ui) { alert(this); });
