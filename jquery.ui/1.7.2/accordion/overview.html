文档内容
概览:
使选择的元素成为Accordion形式的小构件.
译者注: Accordion通常指手风琴式的可拉升菜单.
语法需求:
声明的accordion需要包含成对的头部和内容面板, 如下:
<div id="accordion">
<h3><a href="#">First header</a></h3>
<div>First content</div>
<h3><a href="#">Second header</a></h3>
<div>Second content</div>
</div>
如果需要用其他元素作为header, 可以通过适当的选择器指定header选项, 比如: header: 'a.header', 内容元素必须总是header的下一个元素.
如果在内容中使用了<a />元素, 并且, header同样使用了<a />, 那么就需要增加一个样式并应用到作为header的<a />上, 以辨别他们.
使用activate(Number)方法去改变当前活动的内容部分.
注意: 如果想要一次打开多个部分, 那么不要使用accordion.
一个accordion构件不允许同时打开多个内容面板. 如果你看到一些允许同时打开多个内容面板的构件, 请不要使用它, 通常仅仅用几行jquery代码就可以实现了, 比如:
jQuery(document).ready(function(){
$('.accordion .head').click(function() {
$(this).next().toggle();
return false;
}).next().hide();
});
或者带动态效果的:
jQuery(document).ready(function(){
$('.accordion .head').click(function() {
$(this).next().toggle('slow');
return false;
}).next().hide();
});
