Size Effect

Categories: Effects

Size Effect

Description: 将元素的尺寸设置到一个指定的宽度和高度。

  • size

    • to
      Type: Object
      改变后高和宽。
    • origin (default: [ "top", "left" ])
      Type: Array
      特效起始的方向。
    • scale (default: "both")
      Type: String
      确定元素的哪个区域将被缩放: "both", "box", "content"。Box值缩放元素的border和padding; content值缩放元素内部的所有内容。

Example:

使用尺寸特效来改变div的尺寸。

<script>
$( document ).click(function() {
  $( "#toggle" ).effect( "size", {
    to: { width: 200, height: 60 }
  }, 1000 );
});
</script>
