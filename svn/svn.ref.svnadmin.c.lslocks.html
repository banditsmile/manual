名称

svnadmin lslocks — 打印所有锁定的描述。

概要

svnadmin lslocks REPOS_PATH

描述

打印版本库所有锁定的描述。

选项

无

例子

This lists the one locked file in the repository at /var/svn/repos:

$ svnadmin lslocks /var/svn/repos
