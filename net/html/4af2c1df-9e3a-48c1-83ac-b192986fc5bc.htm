下表显示了由默认构造函数返回的值类型的默认值。默认构造函数是通过 new 运算符来调用的，如下所示：

 复制代码
int myInt = new int();

以上语句同下列语句效果相同：

 复制代码
int myInt = 0;

请记住：在 C# 中不允许使用未初始化的变量。
