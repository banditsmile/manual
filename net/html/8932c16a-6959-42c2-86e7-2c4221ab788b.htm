在 :: 运算符前面使用的 global 上下文关键字引用全局命名空间，该命名空间是任何 C# 程序的默认命名空间，未以其他方式命名。有关更多信息，请参见如何：使用命名空间别名限定符（C# 编程指南）。

示例

下面的示例演示如何使用 global 上下文关键字指定在全局命名空间中定义 TestApp 类：

C# 复制代码
class TestClass : global::TestApp { }
