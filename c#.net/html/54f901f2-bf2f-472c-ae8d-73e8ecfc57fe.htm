备注

除了 object 和 string 外，表中的所有类型均称为简单类型。

C# 类型的关键字及其别名可以互换。例如，可使用下列两种声明中的一种来声明一个整数变量：

 复制代码
int x = 123;
System.Int32 x = 123;

若要显示任何 C# 类型的实际类型，请使用系统方法 GetType()。例如，下列语句显示了表示 myVariable 类型的系统别名：

 复制代码
Console.WriteLine(myVariable.GetType());

还可使用 typeof 运算符。
