BuildPath 方法

在已存在路径后追加名称。

object.BuildPath(path, name)

参数

object

必选项。 应为 FileSystemObject 的名称。

path

必选项。 将被追加 name 的并存在的路径。 路径可以是绝对的或相对的，同时不需要指定已有文件夹。

name

必选项。 被追加到已存在的 path 后的名称。

说明

如果需要，BuildPath 方法会在已有路径和名称之间另外再插入一个路径分隔符。

下面的例子说明了 BuildPath 方法的用法。 

function GetBuildPath(path)
{
   var fso, newpath;
   fso = new ActiveXObject("Scripting.FileSystemObject");
   newpath = fso.BuildPath(path, "New   Folder");
   return(newpath);
}
