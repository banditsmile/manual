DriveLetter 属性

返回一个物理上的本地驱动器或者网络共享的驱动器号。只读属性。

object.DriveLetter

其中 object 总是一个 Drive 对象。

说明

如果所指定的驱动器没有与一个驱动器号关联起来，例如，一个没有映射到驱动器号的网络共享，那么 DriveLetter 属性将返回一个长度为 0 的字符串 ("")。

下面这段代码说明了 DriveLetter 属性的用法： 

function ShowDriveLetter(drvPath)
{
   var fso, d, s;
   fso = new ActiveXObject("Scripting.FileSystemObject");
   d = fso.GetDrive(fso.GetDriveName(drvPath));
   s = "Drive " + d.DriveLetter.toUpperCase( ) + ": - ";
   s += d.VolumeName + "<br>";
   s += "Available Space: " + d.AvailableSpace/1024 + " Kbytes";
   return(s);
}
