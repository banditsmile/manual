需要 Function 对象

试图对一个不是 Function 对象调用 Function prototype 方法，或者在函数调用上下文中使用了对象。例如，下面的代码将产生该错误，因为 foo 不是一个函数。

var foo = new Object();  // 创建一个称为 "foo" 的新对象。
var x = foo();  // 试图将 foo 作为一个函数加以调用。

要改正此错误

  • 只对 Function 对象调用 Function prototype 方法。
  • 确保只使用函数调用操作符() 来调用函数。
