Close 方法

关闭打开的 TextStream 文件。

object.Close( );

object 通常为 TextStream 对象的名字。

说明

下面的例子说明了 Close 方法的用法。

var fso;
fso = new ActiveXObject("Scripting.FileSystemObject");
a = fso.CreateTextFile("c:\\testfile.txt", true);
a.WriteLine("This is a test.");
a.Close();
