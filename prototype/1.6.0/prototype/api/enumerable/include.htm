include

include(object) -> Boolean

判断 Enumerable 中是否存在指定的对象，基于 == 操作符进行比较。别名 member。

注意，这不是严格意义上的比较（===，同时比较值和类型），而是等价比较（仅对值进行比较， 若数据类型不同，则进行隐式转换）。

如果需要检测 Enumerable 中是否有一个元素匹配给定的谓词，请使用 any。

样例

$R(1,15).include(10) 
// -> true 
['hello', 'world'].include('HELLO')
// -> false 
[1, 2, '3', '4', '5'].include(3) 
// -> true (== 忽略数据类型)
