inspect

inspect([useDoubleQuotes = false]) -> String

返回该字符串针对调试的字符串表现形式（即用单引号或双引号包括起来，并使用 '\' 对特殊字符进行转义）。

关于 inspect 方法的更多信息，请参见 Object.inspect。

样例

'I\'m so happy.'.inspect(); 
// -> '\'I\\\'m so happy.\''（在一个弹出对话框或控制台中显示为 'I\'m so happy.'） 
'I\'m so happy.'.inspect(true);
// -> '"I'm so happy."'（在一个弹出对话框或控制台中显示为 "I'm so happy."）
