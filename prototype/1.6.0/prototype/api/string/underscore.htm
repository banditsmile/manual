underscore

underscore() -> string

将一个 Camel 形式的字符串转换为以下划线（"_"）分隔的一系列单词。

样例

'borderBottomWidth'.underscore(); 
// -> 'border_bottom_width'

注意

为将一个 DOM style 属性转换为它等价的 CSS 属性，请联合使用使用 String#dasherize 和 String#underscore 方法。 

'borderBottomWidth'.underscore().dasherize(); 
// -> 'border-bottom-width'
