clone
不推荐

clone(source, target[, options]) -> [Number, Number]

从 source 中复制参数 options 指定的位置和/或尺寸的相关信息到 target。

已被 Element#clonePosition 方法取代。

注意：该方法根据 CSS containing block 的规则判定元素的位置信息是否相同，调用该方法后，target 和 source 的位置信息按照上述规则严格匹配。

选项

名称默认值描述
setLeft true 克隆 source 的 CSS 属性 left 到 target。
setTop true 克隆 source 的 CSS 属性 top 到 target。
setWidth true 克隆 source 的 CSS 属性 width 到 target。
setHeight true 克隆 source 的 CSS 属性 height 到 target。
offsetLeft 0 数值，指定 target 的 CSS 属性 left 的起始偏移值。
译注：target 最终的 CSS 属性 left 等于 source 的 left 值加上此处的 offsetLeft 值。
offsetTop 0 数值，指定 target 的 CSS 属性 top 的起始偏移值。
译注：target 最终的 CSS 属性 top 等于 source 的 top 值加上此处的 offsetTop 值。

译注：options 是包含名值对的 hash 对象，如果想调用该方法，但又不希望改变 target 的 width 和 height，则 options 应设置为 {setWidth:false, setWidth:false}。
