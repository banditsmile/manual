unset
1.6

unset(key) -> value

删除 hash 中的键，并返回键对应的值。

样例

var h = new Hash({ a: 'apple', b: 'banana', c: 'coconut' }); 
h.unset('a'); 
// -> 'apple' 
h.inspect(); 
// -> #<Hash:{'b': 'banana', 'c': 'coconut'}> 
h.unset('d'); 
// -> undefined 
h.inspect(); 
// -> #<Hash:{'b': 'banana', 'c': 'coconut'}>
