toObject() -> Object

克隆 hash 內部的 Object 并返回。译注：Hash 在内部使用一个单独的 Object 保存“键/值”对。

样例

var h = new Hash({ a: 'apple', b: 'banana', c: 'coconut' }); 
var obj = h.toObject(); 

Object.inspect(h); 
// -> #<Hash:{'a': 'kiwi', 'b': 'banana', 'c': 'coconut'}> 
Object.inspect(obj); 
// -> [object Object] 
obj.a = 'orange'; 
h.get('a'); 
// -> 'apple'
