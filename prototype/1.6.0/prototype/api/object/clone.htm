clone

Object.clone(obj) -> Object

使用浅拷贝克隆传入的对象（复制所有原始的属性到结果中）。

请注意是浅拷贝，不是深拷贝。

样例

var o = { name: 'Prototype', version: 1.5, authors: ['sam', 'contributors'] }; 
var o2 = Object.clone(o);
 
o2.version = '1.5 weird'; 
o2.authors.pop();

o.version 
// -> 1.5 
o2.version 
// -> '1.5 weird' 
o.authors 
// -> ['sam'] 
// 噢！浅拷贝！
