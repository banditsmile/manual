setValue

setValue(element, value) -> HTMLElement

设置表单控件的值。对于多选框，参数 value 应传入一个数组，其它的控件传入字符串。

译注：官方帮助手册中并没有该方法相关的文档，但在 Prototype 的程序源码中可以找到这个方法， 并且，官方手册中列出了 getValue，所以在此补上该方法的文档。

样例

$('inputid').setValue('请选择……')
//当然，也可以这样：
Form.Element.setValue('inputid', '请选择……')

//如果是多选的 SELECT
$('multiSelect').setValue(['音乐', '读书'])

注意

如果参数 element 是一个未知的 ID，将会抛出一个异常："element has no properties"。
