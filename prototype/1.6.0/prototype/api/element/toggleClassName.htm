toggleClassName

toggleClassName(element, className) -> HTMLElement

切换 element 的 CSS className（在有和无该类名称间切换），并返回元素本身。

样例

<div id="mutsu" class="apple"></div> 
$('mutsu').hasClassName('fruit'); 
// -> false 
$('mutsu').toggleClassName('fruit');
// -> element 
$('mutsu').hasClassName('fruit'); 
// -> true
